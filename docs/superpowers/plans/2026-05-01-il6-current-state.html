1. 한 줄 요약 + 코어 루프

"끝없는 겨울. 마지막 마을의 마지막 수호자." 한국어 판타지 설원 탑다운 생존 + 베이스 디펜스. 낮에는 자원 채집 + 동료 영입 + 마을 확장(펜스도 자동으로 자라남), 밤에는 좀비 웨이브 사수.

코어 루프

  1. 낮 (Day) — 자원 채집(나무/돌/고기), 동물 사냥, 떠돌이 영입, 건물 짓기 + 자원으로 모닥불 연료 보충, 동료 레벨업.
  2. 저녁 (Evening, 5s) — 화면 검게 페이드 → 마을로 자동 텔레포트 (플레이어 + 동료 + 펫 모두).
  3. 밤 (Night) — 좀비 웨이브 무한 보충, 25킬마다 보스 강제, Day 4+ 부터 원거리 Archer 좀비. 마을 한기는 모닥불 연료가 살아있을 때만 막아짐.
  4. 새벽 (Dawn, 5s) — 골드빛 시네마틱 + "☀ 아침이 밝았다", EndNight 모든 좀비 청소, 풀 HP 회복, 건물 +25% HP, 새 NPC 1명 스폰, 자동 저장.

2. 사이클 시스템

  • 기본 페이즈 길이: 낮 40s, 밤 40s, 저녁 5s, 새벽 5s.
  • 레벨 스케일: Day/Night 만 1 + 0.25 × (Level-1) 곱연산. Lv 1 = 40s, Lv 5 = 80s, Lv 10 = 140s. 저녁/새벽 고정.
  • 페이즈 길이 잠금: DayNightController._phaseLockedDuration 캐싱 — 진행 중 레벨업해도 현재 페이즈 길이는 안 변함.
  • 밤 진입 안전망: NightController.Update 가 Cycle.Phase ↔ CurrentPhase 동기화 (이벤트 누락 케이스 보장). Zombie.Update 도 본인 phase != Night 면 자폭.

3. 마을 + 건물 (10종)

고정 구조

게임 시작 시 VillageStarter.SpawnStarterVillage 가 마을 중심에 모닥불 + ±5u 사각 펜스 + 남쪽 게이트 (Door, Player+Companion 통과). 건물을 지을 때마다 펜스 사각이 자동으로 확장 — 비-펜스 건물 1채당 +0.4u, 최대 14u 까지. 새 외곽에 fence ring + 게이트 생성, 0.4u 안에 기존 펜스 있으면 중복 스킵.

  2. 슬롯 클릭 → _pendingBuildKind 설정. 마우스에 골드 + 커서 + 안내 배너.
  3. 좌클릭 = 자원 즉시 차감 + ConstructionSite 스폰. 우클릭/ESC = 취소.
  4. ConstructionSite 8초 기본, 반경 2.5u 안 비-Hiding/Working 동료 1명당 +1× 속도 (단독 8s, 동료 2명 ~3s).
  5. 완료 시 실제 건물 스폰 + 비-펜스면 OnBuildingAdded → 펜스 ring 확장.

비용 점증 + 마을 성장도 HP

  • 건물 HP: 1 + 0.12 × 비-펜스 건물 수 (캡 ×3) — 마을이 커질수록 새 건물 더 튼튼.
  • 펜스 HP: 1 + 0.5 × 망루 수

4. 유닛 — 플레이어 / 동료 / 펫

플레이어

  • HP 100, MoveSpeed 5, 자동 공격 (모드 따라 근접/원거리)
  • HP 회복: 4초마다 +1, 새벽 풀회복
  • 모드 탭 [1] ⚔ 근접 / [2] 🏹 원거리 / [3] 🏠 건축
  • 이동 시 발자국 눈 퍼프, 피격 시 화면 빨강 비네트 + 카메라 셰이크

동료 (Companion) 개별 진행 추가

  • HP 50, IsCombat 따라 자동 공격. 사거리 5u, 시야 9u.
  • Stance: 👣 Follow / 🛡 Hold / ⚔ Aggressive 토글
  • 포메이션: 둘러싸기 X, 플레이어 facing 기준 종대 한 줄에 2명씩
  • follow 우선순위: 시야 밖 → 따라옴 / 시야 안 + 적 → 추격 / 둘 다 마을 안 → 앵커 / 다른 쪽 → 문 통과 2단계
  • 채집 중엔 전투 X, 단 피격 시 즉시 중단 후 반격
  • 자동 대피 (1초 폴링): 눈보라 또는 비전투+적 시야 → 가까운 비-펜스/바리게이트 건물에 숨음
  • 망루 부스트: 3.5u 내 dmg ×1.5
  • 개별 XP/레벨: 좀비 처치 시 가장 가까운 동료에 +1 XP. XpToNext = 4 + Level*3. 레벨당 dmg +10%, MaxHp +8%, 레벨업 시 +10 HP 회복 + "Lv N!" 토스트.
  • 장비 슬롯: SharpenedBlade(+30% dmg) / Crossbow(+20% dmg) / ThickFur(+25% HP) / ToolKit (현재 enum 만, 부착 UI 는 future work).
  • 마을 수용 한도: FreeCapacity(12) + Houses × 4. 일일 영입 한도 X — MaxAliveNpcs(3) + Interval(30s) 자연 캡.
  • 먼 거리(8u 이상) 시 collider OFF — 정체 방지

펫 (Pet)

  • SummonDog 룬 → 갈색 둥근 펫. 근접 5+lvl×3 dmg, 1.4u, 0.55s.
  • SummonHawk 룬 → 노란 삼각 펫. 매도 근접, 5+lvl×3 dmg, 1.2u, 0.7s.
  • 둘 다 RB2D + collider — 다른 유닛처럼 충돌. Player 주변 궤도, 좀비 자동 추격 슬래시.

5. 적 — 좀비 + 보스

좀비 (Zombie)

  • 밤에만 스폰. 펜스 사각(동적 ±halfSize + 1u 여유) 안이면 외부로 강제 push.
  • 일자별 강화: HP × 1 + 0.12 × (day-1), dmg +(day-1), speed × 1 + 0.02 × (day-1) (캡 1.5×).
  • 변종:
    • Day 3+ Fast (오렌지, 1.6× speed, 60% HP)
    • Day 7+ Tank (보라, 0.6× speed, 220% HP, +4 dmg)
    • Day 4+ Archer 신규 — 어두운 녹색, HP 80%, 1.5s 쿨로 7u 사거리 녹색 화살 (6+(day-1) dmg). 마을 안에서 안전하게 때리니 플레이어가 펜스 밖으로 나가서 처치해야 함.
  • 경로 차단 시 건물 부수기 (Linecast → Building 코사이더 우선 공격)
  • 상태이상: Poison(누적 DoT), Slow, Lightning chain, Detonate

보스 대폭 강화

  • Day 5/10/15 BossNight + 임의 밤 KillsThisNight ≥ 25 강제 소환
  • HP 100 + day × 100 — Day 5 = 600, Day 10 = 1100, Day 15 = 1600
  • Damage 보너스 18 + (day-1) × 2 — Day 5 = 26, Day 10 = 36
  • 3초 카운트다운 ⚠ 배너 + 카메라 셰이크 + 음악 트리거

웨이브 밀도 ↑

  • Day 5 웨이브: 18→32, Day 10: 33→62
  • 활성 0+대기 0 시 무한 보충 Base + KillsThisNight/5

9. 모닥불 연료 + 마을 한기 ⚡ 핵심 신규

이번 갱신의 가장 큰 시스템. 마을이 단순 방어 거점이 아니라 지속적 관리 대상이 됨.

모닥불 연료

  • 각 모닥불은 Fuel (max 100, BurnRate 1/s) — 약 100초 지속.
  • 0 도달 시: 좀비 화상 오라 정지, 스프라이트 35% 어두워짐.
  • 월드 버튼 "🔥 장작 +30 (1W)" 으로 보충. 100% 시 버튼 미표시.

마을 한기 데미지

  • SnowfieldController 가 4초마다 마을 펜스 안 (동적 halfSize) 검사
  • CampfireAura.VillageHasHeat = 활성 모닥불 1개 이상 안에 있으면 true
