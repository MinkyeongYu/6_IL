1. 게임 개요

눈보라 치는 판타지 설원, 낮에는 사냥꾼·밤에는 마을 방어자. 영속으로 쌓이는 마을과 잃으면 돌아오지 않는 동료들 사이에서, 얼마나 오래 버틸 수 있는가.

장르: 뱀서류 + 낮/밤 서바이벌 + 타워디펜스 + 핵앤슬래시

2. 핵심 게임플레이 루프

하루 구조

하루 = 낮(Day) → 저녁(Evening) → 밤(Night) → 새벽(Dawn)
단계길이행동
낮120초설원 탐험, 자원 수집, 사냥
저녁10초전환 경고, 마을 귀환
밤90초좀비 웨이브 방어
새벽10초정리, 다음 날 준비

4가지 게임플레이 기둥

1. 낮의 사냥꾼 — 설원 탐험, 자원 수집, 사냥
2. 밤의 성주 — 모닥불 방어, 바리게이트 배치, 좀비 웨이브 격퇴
3. 영속하는 마을, 잃어버리는 동료들 — 마을은 성장, 동료는 영구 사망
4. 초대형 눈보라 — 드문 극한 이벤트 (Phase 3 예정)

3. 현재 구현 상태 (v0.2.0)

플레이어

  • WASD/방향키 이동 (속도 120px/s)
  • 근접 자동 공격 (롱소드: 데미지 25, 사거리 48px, 쿨다운 600ms)
  • HP 100, 사망 시 게임오버 화면
  • R키로 재시작

설원 탐험

나무 — F키 벌목, 나무 3개 획득
바위 — F키 채광, 돌 2개 획득
사슴 — 도망 AI, 사냥 시 고기 2개 드랍
수집 — 진행 바 표시

자원 시스템

자원획득상태
나무 (Wood)벌목구현
돌 (Stone)채광구현
고기 (Meat)사슴 사냥구현
철 (Iron)심경 광맥미구현
식량 (Food)밭미구현
눈꽃 결정보스/눈보라미구현

마을 방어

  • 모닥불 (마을 중심): 반경 128px 내 좀비에 5 DPS
  • 바리게이트 12개 자동 배치 (동서남북 방향)
  • HP 200, 좀비가 공격하여 파괴 가능

좀비 웨이브

  • 밤마다 3개 웨이브
  • 웨이브당 좀비 수: 8 + (day-1)×4 + (wave-1)×2
  • 마을 외곽 원형 스폰
  • 좀비 스탯: HP 60, 데미지 10, 방어 2, 속도 40px/s

시야 시스템

  • 낮: 넓은 시야
  • 밤: 좁은 시야 (어두운 사각형 마스크)

UI/HUD

  • 좌상단: 자원 현황 (나무/돌/고기)
  • 우상단: 일차/단계/남은시간
  • 좌하단: HP 바
  • 우하단: 웨이브 정보 (밤)
  • 게임오버: 생존 일수, 처치 수, R키 재시작
