# Ini's Land 기획서

> 끝없는 겨울 속에서 낮에는 생존 자원을 모으고, 밤에는 마을을 지키는 설원 생존 액션 게임.

**장르:** 로그라이트 생존 액션 + 마을 방어 + 동료 성장
**현재 버전:** v0.2.0
**권위 구현:** Unity 2022.3.62f3 LTS, C#, Unity 2D/URP
**권위 빌드:** `IL6.EditorBuild.BuildScript.BuildWindows`

---

## 1. 핵심 문제

짧은 세션 안에서 탐험, 채집, 성장, 방어가 자연스럽게 순환하는 겨울 생존 게임을 목표로 한다. 플레이어는 낮에 설원을 탐험하며 자원과 동료를 확보하고, 밤에는 좀비 웨이브와 혹한을 버티며 마을의 핵심 시설을 지켜야 한다.

---

## 2. 코어 루프

```text
낮 40초+ : 설원 탐험, 나무/돌/고기/식량 채집, 동물 사냥, 동료 영입, 건물 건설/업그레이드
저녁    : 플레이어와 동료 귀환, 마을 방어 준비
밤 40초+: 좀비 웨이브, 모닥불/화로/울타리/동료 방어, 혹한과 시야 제한
새벽    : HP 회복, 건물 자동 수리, NPC 스폰, 저장
```

밤 길이는 날짜가 지날수록 증가하며, 25일 단위로 보스 위협이 등장한다.

---

## 3. 구현된 주요 시스템

- 플레이어 이동, 근접/원거리 무기 전환, 레벨업 선택지
- 낮/밤 사이클, 밤 시야 마스크, 모닥불/화로/대장간 온도 오라
- 좀비 웨이브, 보스형 적, 동물 사냥, 늑대 안전 스폰 제한
- 동료 영입 연출, 영입 중 일시정지, 동료 이름/특성/장비 강화
- 인구 증가에 따른 식량 소비, 부부/임신/7일째 출산 시스템
- 동료 수가 적을 때 마을 근처 NPC 스폰 보정
- 작물 선택, 작물 성장/수확/시듦, 저온 생존률/수확량 차등
- 울타리 자동 확장, 낮 시간 울타리 일괄 수리와 동료 수리 작업
- 중기 목표, 건물 레벨업, 단계별 자원 소모형 성장 구조
- 눈 파티클과 눈보라 강화 연출
- HUD 아이콘, 작물 아이콘, 발견물/건물 식별 스프라이트 적용

## 5. 작물 시스템

작물은 기온과 건물 레벨의 영향을 받는다.

| 작물 | 특징 |
| --- | --- |
| Potato | 기본 작물. 추위에 비교적 안정적이고 평균 수확량이 좋다. |
| Turnip | 저온 생존률이 높지만 수확량이 낮다. |
| Wheat | 수확량이 높지만 추위와 눈보라에 취약하다. |

작물은 성장 중, 수확 가능, 시든 상태를 월드 스프라이트와 색상으로 표시한다. 시든 밭은 컨텍스트 액션으로 정리할 수 있다.

---

## 6. 동료와 마을 인구

- 동료는 영입 시 이름과 특성을 받는다.
- 동료 영입 대화는 바스트샷 연출처럼 표시되고, 영입 중에는 게임 시간이 정지된다.
- 동료가 3명 이하일 때는 늑대가 스폰되지 않는다.
- 동료가 적을수록 마을 근처에서 영입 가능한 NPC가 더 자주, 더 가까이 등장한다.
- 성인 남녀는 같은 파트너와만 아이를 낳으며, 임신 후 7일째 아침에 아이가 태어난다.
- 인구가 늘수록 일일 식량 소비량도 증가한다.

---

## 7. 리소스 적용 기준

- Unity 런타임 리소스는 `Assets/Resources/`와 Unity import `.meta`를 기준으로 관리한다.
- 이미 프로젝트에 들어온 외부/승인 리소스를 우선 적용한다.
- PowerShell `System.Drawing`으로 만든 저품질 절차형 아트는 새 게임 리소스로 사용하지 않는다.
- 스프라이트가 없을 경우 단색 fallback만 남기지 않고, 가능한 기존 HUD/작물/프랍 아이콘을 재사용한다.

## 8. 기술 스택

- **엔진:** Unity 2022.3.62f3 LTS
- **언어:** C#
- **렌더링:** Unity 2D + Universal Render Pipeline
- **입력/UI:** Unity runtime scripts + IMGUI HUD
- **빌드:** `IL6.EditorBuild.BuildScript.BuildWindows`
- **출력:** `Build/Windows/IL6.exe`, 루트 배포용 `6IL_v0.2.0_portable.exe`
- **검증:** Unity batch compile/build logs, 런타임 실행 확인, git diff/status 확인
