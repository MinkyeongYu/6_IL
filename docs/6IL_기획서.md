## 1. 게임 개요

### 한 줄 비전
눈보라 치는 판타지 설원, 낮에는 사냥꾼·밤에는 마을 방어자. 영속으로 쌓이는 마을과 잃으면 돌아오지 않는 동료들 사이에서, 얼마나 오래 버틸 수 있는가.

### 장르
뱀서류 + 낮/밤 서바이벌 + 타워디펜스 + 핵앤슬래시, 판타지 중세 설원판

### 기술 스택
| 항목 | 선택 |
|------|------|
| 게임 엔진 | Phaser 3 |
| 언어 | TypeScript (strict) |
| ECS | bitecs |
| 빌드 | Vite |
| 테스트 | Vitest |
| 데스크톱 래퍼 | Electron + electron-builder |

## 2. 핵심 게임플레이 루프

### 하루 구조
```
하루 = 낮(Day) → 저녁(Evening) → 밤(Night) → 새벽(Dawn)
```

| 단계 | 길이 | 행동 |
|------|------|------|
| 낮 | 120초 | 설원 탐험, 자원 수집, 사냥 |
| 저녁 | 10초 | 전환 경고, 마을 귀환 |
| 밤 | 90초 | 좀비 웨이브 방어 |
| 새벽 | 10초 | 정리, 다음 날 준비 |

### 4가지 게임플레이 기둥
1. **낮의 사냥꾼** — 설원 탐험, 자원 수집, 사냥
2. **밤의 성주** — 모닥불 방어, 바리게이트 배치, 좀비 웨이브 격퇴
3. **영속하는 마을, 잃어버리는 동료들** — 마을은 성장, 동료는 영구 사망
4. **초대형 눈보라** — 드문 극한 이벤트 (Phase 3 구현 예정)

## 3. 현재 구현 상태 (v0.2.0)

### 구현 완료 (Phase 1)

#### 플레이어
- WASD/방향키 이동 (속도 120px/s)
- 근접 자동 공격 (롱소드: 데미지 25, 사거리 48px, 쿨다운 600ms)
- HP 100, 사망 시 게임오버 화면
- R키로 재시작

#### 설원 탐험
- 나무 (초록 사각형) — F키로 벌목, 나무 3개 획득
- 바위 (회색 사각형) — F키로 채광, 돌 2개 획득
- 사슴 (갈색 사각형) — 도망치는 AI, 사냥 시 고기 2개 드랍
- 수집 시 진행 바 표시

#### 자원 시스템
| 자원 | 획득 방법 |
|------|----------|
| 나무 (Wood) | 벌목 |
| 돌 (Stone) | 채광 |
| 철 (Iron) | 미구현 |
| 고기 (Meat) | 사슴 사냥 |
| 식량 (Food) | 미구현 |
| 눈꽃 결정 (Frostbloom) | 미구현 |

#### 마을 방어
- 모닥불 (마을 중심): 반경 128px 내 좀비에 5 DPS
- 바리게이트 12개 자동 배치 (동서남북 방향)
- HP 200, 좀비가 공격하여 파괴 가능

#### 낮/밤 사이클
- Day → Evening → Night → Dawn 4단계 순환
- 이벤트 버스로 전환 알림
- HUD에 현재 단계/남은 시간 표시

#### 좀비 웨이브
- 밤마다 3개 웨이브
- 웨이브당 좀비 수: 8 + (day-1)×4 + (wave-1)×2
- 마을 외곽 원형 스폰
- 좀비 스탯: HP 60, 데미지 10, 방어 2, 속도 40px/s
- 플레이어 추적 AI

#### 시야 시스템
- 낮: 넓은 시야
- 밤: 좁은 시야 (어두운 사각형 마스크)

#### UI/HUD
- 좌상단: 자원 현황 (나무/돌/고기)
- 우상단: 일차/단계/남은시간
- 좌하단: HP 바 (초록→빨강)
- 우하단: 웨이브 정보 (밤)
- 게임오버 화면: 생존 일수, 처치 수, R키 재시작

## 5. 아키텍처

### ECS (bitecs)
```
Components: Position, Velocity, Health, Combat, PlayerTag, ZombieTag, DeerTag, ResourceNode, Warmth
Systems: movementSystem, combatSystem, zombieAiSystem, deerAiSystem, deathSystem
```

### 주요 파일 구조
```
src/
├── main.ts                      # Phaser 부트스트랩
├── config/
│   ├── balance.ts               # 게임 밸런스 상수
│   ├── weapons.ts               # 무기 데이터
│   └── enemies.ts               # 적 데이터
├── scenes/
│   ├── boot.ts                  # 부트 씬
│   ├── preload.ts               # 텍스처 생성
│   └── game.ts                  # 메인 게임 씬
├── ecs/
│   ├── components.ts            # ECS 컴포넌트 정의
│   ├── world.ts                 # GameWorld 타입 + 팩토리
│   └── systems/
│       ├── movement-system.ts   # 위치 업데이트
│       ├── combat-system.ts     # 자동 전투
│       ├── zombie-ai-system.ts  # 좀비 추적 AI
│       ├── deer-ai-system.ts    # 사슴 도망 AI
│       └── death-system.ts      # 사망 처리
├── gameplay/
│   ├── combat.ts                # 데미지 계산
│   ├── day-night-cycle.ts       # 낮/밤 상태 머신
│   ├── resource-manager.ts      # 자원 관리
│   ├── wave-spawner.ts          # 웨이브 스포너
│   └── village-grid.ts          # 마을 건물 그리드
├── events/
│   ├── event-bus.ts             # 이벤트 버스
│   └── types.ts                 # 이벤트 타입
├── input/
│   └── input-adapter.ts         # 입력 추상화
├── ui/
│   ├── hud.ts                   # 인게임 HUD
│   └── game-over.ts             # 게임오버 화면
└── util/
    ├── rng.ts                   # 시드 기반 RNG
    └── sprite-registry.ts       # ECS↔Phaser 스프라이트 매핑
```
