## § 1. 비전과 핵심 기둥

### 1.1 한 줄 비전

눈보라 치는 판타지 설원, 낮에는 사냥꾼·밤에는 마을 방어자. 영속으로 쌓이는 마을과 잃으면 돌아오지 않는 동료들 사이에서, 얼마나 오래 버틸 수 있는가.

### 1.2 장르 슬로건

> "뱀서류 + 낮/밤 서바이벌 + 타워디펜스 + 핵앤슬래시, 판타지 중세 설원판."

### 1.3 4가지 게임플레이 기둥

1. **낮의 사냥꾼** — 제한 시간 내 설원 탐험. 동료 영입, 자원 수집, 거대동물 사냥. 시야/안개가 긴장감을 만들고, 대담함과 귀환 타이밍을 저울질.
2. **밤의 성주** — 모닥불로 불빛을 밝히고, 그리드에 바리게이트를 깔고, 영입한 동료들과 함께 좀비 무리를 새벽까지 막아냄. 뱀서류의 자동 공격 + 타워디펜스의 전략 배치.
3. **영속하는 마을, 잃어버리는 동료들** — 마을은 계속 성장 (영속). 하지만 동료는 영구 사망. 한 번 죽으면 돌아오지 않음 → "누구를 밖에 데리고 갈지"에 무게감.
4. **아주 드문 초대형 눈보라** — 밤의 확장 이벤트. 좀비 + 극한 추위가 동시. 모닥불/집 밖은 지속 대미지. 모든 시스템이 교차하는 절정.

### 1.4 타깃 플레이어 여정

- Day 1~5: 생존 학습
- Day 10~30: 마을 안정화, 동료 누적
- Day 50+: 장비 빌드 실험, 최고 기록 경신
- 한 마을당 10~30시간
- 종료 조건: 전멸(마을 궤멸)만. 엔딩은 없음 (무한 서바이벌)
- 전멸 시 점수 기록 + 일부 메타 해금 → 새 마을 시작

### 1.5 타깃 플랫폼

| 우선순위 | 플랫폼 | 방식 |
|---|---|---|
| 1차 | 웹 (itch.io, 자체 사이트) | Phaser 3 HTML5 |
| 2차 | Steam (데스크톱) | Tauri 래핑 |
| 3차 | 모바일 (iOS/Android) | Capacitor 래핑 |

## § 2. 기술 스택과 아키텍처

### 2.1 스택

| 계층 | 선택 | 역할 |
|---|---|---|
| 런타임 | 브라우저 (Chromium 기반) | 1차 타깃 |
| 게임 엔진 | Phaser 3 | 2D 렌더, 신 관리, 입력 |
| 언어 | TypeScript (strict) | 타입 안전, 리팩토링 안정성 |
| 빌드/개발 서버 | Vite | HMR, 빠른 개발 루프 |
| 데스크톱 래퍼 | Tauri | Steam 빌드용 실행파일 (~5MB) |
| 모바일 래퍼(3차) | Capacitor | iOS/Android 포장 |
| ECS | 자체 경량 ECS (`bitecs` 후보) | 수천 엔티티 성능 |
| 맵 에디터 | Tiled | 타일맵·콜리전·스폰 포인트 |
| 저장 | localStorage (웹), 파일(Tauri) | 마을 영속 데이터 |
| 오디오 | Howler.js (또는 Phaser 내장) | BGM/SFX 믹싱 |
| 에셋 파이프라인 | 수동 + sprite atlas (TexturePacker 무료 또는 free-tex-packer) | 스프라이트 묶음 |

**향후 이식**: Unity 이식은 장기 로드맵. 현재 설계는 Phaser 기준.

### 2.3 설계 원칙

- **ECS로 전투 물량 처리** — 좀비 수백 마리가 나오는 밤을 대비. Phaser의 GameObject 기반 로직은 UI/건물 등 소수 오브젝트에만 사용.
- **데이터 중심(Data-Driven)** — 적, 무기, 건물, 동료 종류, 룬 모두 `src/content/`의 JSON/TS 데이터 파일로 선언. 코드 변경 없이 밸런스 조정.
- **신 경계로 모드 분리** — `SnowfieldScene`(낮)과 `VillageScene`(밤) 분리. 각각 다른 시스템 조합 구동.
- **시뮬 틱 vs 렌더 틱 분리** — 시뮬레이션은 고정 dt(30Hz 또는 60Hz), 렌더는 프레임레이트 따라감. 퍼포먼스/결정성 확보.
- **모바일 고려** — 입력 추상화 레이어(`InputAdapter`) — 키보드/게임패드/터치를 같은 인터페이스로. 해상도는 스케일링으로 처리.

### 2.4 저장 데이터 구조 (영속)

```ts
type SaveFile = {
  version: number;                    // 스키마 버전 (마이그레이션용)
  village: VillageState;              // 건물 레벨, 배치
  resources: Record<ResourceId, number>;
  companions: CompanionState[];       // 살아있는 동료만
  metaUnlocks: string[];              // 다음 마을 시작 시 보너스
  highScore: {
    day: number;
    kills: number;
    bossesKilled: number;
    megaBlizzardsSurvived: number;
    companionsLost: number;
    finalScore: number;
  };
  currentDay: number;
  weatherRng: number;                 // 재현 가능한 날씨 시드
};
```

### 2.5 퍼포먼스 타겟

- 밤 피크(좀비 500~800마리 동시): 60 FPS on 중급 데스크톱 브라우저
- 모바일 타깃: 최적화 후 중급 폰에서 30~60 FPS, 좀비 수 자동 조정
- 초기 로드: 웹에서 1회 <10MB 번들 (atlas 압축 포함)

## § 3. 게임 루프 상세

### 3.1 시간 구조

```
하루(One Day) = 낮(Day Phase) + 저녁 전환 + 밤(Night Phase) + 새벽 전환
```

| 단계 | 길이(목표) | 플레이어 주요 행동 |
|---|---|---|
| 낮 | 8~10분 | 설원 탐험·영입·사냥·자원 수집. 귀환 타이밍 선택. |
| 저녁 전환 | 30초 | 종탑 종소리. 모든 NPC/동료 자동 귀환. 건물/바리게이트 재배치 가능. |
| 밤 | 5~7분 | 좀비 웨이브 방어. 새벽까지 생존. |
| 새벽 전환 | 30초 | 좀비 잔당 소멸, 시체 자원 회수, 다음 날 준비. |

하루 총 길이: 약 14~18분. 한 세션 1~2시간이면 5~8일 진행 가능.

### 3.2 낮 (Snowfield)

**플레이어 목표 (비강제)**
- 자원 수집 (나무, 돌, 약초, 고기)
- 동료 영입 (설원에 서 있는 NPC에게 말 걸기)
- 거대동물 사냥 → 많은 고기·가죽·희귀 재료
- 룬/축복 아이템 획득 (이번 밤에만 유효)

**시야 시스템**
- 플레이어 중심 원형 시야 (반경 ~10타일)
- 밖은 안개 + 그림자 아이콘(희미한 실루엣)
- 높은 지형·망루로 시야 확장
- 거대동물은 발자국/소리 큐로 위치 힌트

**적·동물 구역화**

| 구역 (마을 중심→) | 내용 |
|---|---|
| 근교 (반경 1) | 사슴, 토끼 등 약한 수동 동물 + 드문 나무/돌. 안전. 초반 생존 구역. |
| 중경 (반경 2) | 늑대 무리, 더 많은 자원, 룬 조각 간혹. Day 4~6부터 나타남. |
| 심경 (반경 3+) | 서리 정령, 서리 트롤, 철 광맥, 거대동물(보스급), 희귀 룬. Day 8+ 도전 구역. |

- Day 1~2 설원 전투는 매우 제한적. 대부분 채집·사냥 중심.
- 사슴·토끼는 접근 시 도망 → 활로 원거리 처치 또는 기습 접근 필요.
- 거대동물은 심경에서만 (현무형 거대 사슴, 서리 매머드 등).

**수집 액션(시간 코스트)**

| 행동 | 도구 | 액션 | 시간 |
|---|---|---|---|
| 벌목 | 도끼 또는 근접 무기 | 나무에 연속 타격 → 쓰러짐 | 3~5초/그루 |
| 채광 | 곡괭이 또는 둔기 | 바위 노드 파괴 | 5~8초/노드 |
| 사냥 | 활 추천 / 근접 시 뒷치기 | 동물 접근 → 공격 → 도축 | 2초 도축 + 추적 시간 |

→ "이 8~10분 낮에 나무를 벨까, 사슴을 쫓을까, 더 깊이 들어갈까" 저울질이 핵심 재미.

**귀환 타임아웃**
- 종 울림 후에도 설원에 남아 있으면 → 밤이 설원에서 시작됨 → 안전 지대 없음 → 곧 사망
- 페널티로 "야외 밤 생존" 미니 도전 모드(자원 없이 살아 돌아가면 희귀 보상)

### 3.3 밤 (Village Defense)

**씬 전환**
- 카메라가 마을 중심으로 귀속. 설원 지역은 닫힘.
- 밤 시작 전 30초 건설/재배치 페이즈 — 바리게이트 설치, 동료 배치 지점 지정, 모닥불 연료 보충.

**좀비 웨이브**
- 밤 동안 3~5개 파동, 점점 강해짐
- 웨이브 간 10~20초 휴식
- 방향: 마을 외곽 스폰 포인트 2~4곳에서 진입 (마을 배치에 따라 다름)

**전투**
- 플레이어: 수동 조작 + 뱀서류식 자동 공격 (장비/룬 기반)
- 동료: AI, 배치 지점 주위 일정 반경 방어 (또는 플레이어 추종)
- 모닥불: 반경 내 좀비 지속 화염 대미지 + 시야 제공 + 동료 공격력 버프
- 바리게이트: 좀비가 부수는 동안 공격 유도 + 체력 있음 + 부서지면 교체 필요
- 종탑: 레벨업 시 원거리 투사체 자동 발사 (서포트 타워)

**보스 밤**
- 주기적: Day 5·10·15·...마다 고정 보스 등장 (한 마리, 일반 웨이브 사이)
- 초기 보스 종류 (3~5개):
  - 거대 서리 좀비 (체력·물리 공격)
  - 언데드 겨울기사 (근접 + 방패, 동료 타겟 유도)
  - 얼어붙은 거인 (광역 얼음 브레스, 건물 파괴 특화)
  - 서리 리치 (원거리 스킬, 좀비 소환, 건물 우선 타겟)
  - *초대형 눈보라 전용 네임드 보스 별도*
- 처치 시: 희귀 룬 재료, 메타 해금 포인트, 조각난 진화 무기 키 등

### 3.4 초대형 눈보라 밤 (드문 이벤트)

**트리거**
- 주기: 7~15일 랜덤 (시드 기반, 예측 힌트 있음)
- 힌트: 전날 낮 후반에 "차가운 바람이 거세진다" 메시지, 동물들 사라짐

**경고 신호 (사이렌 없음, 중세 분위기)**
- 저녁 전환 시: 종탑 종소리 + 하늘 어두워짐 + 까마귀 소리 + 30초 카운트다운

**효과**
- 일반 좀비 수 2배 + 네임드 보스
- **체온 시스템 활성화**: 모닥불/건물 반경 밖에 있으면 초당 누적 대미지 (HP 지속 감소)
- 시야 축소 (안개 강해짐, 낮의 50% 수준)
- 바리게이트 내구도 감소 (추위로 나무가 약해짐)

**보상**
- 생존 시: 마을 번영도 보너스, 희귀 재료, 메타 해금 포인트 대량

### 3.5 전체 세션 흐름 예시

```
Day 1-3  : 튜토리얼형 안정기, 기초 건물 해금
Day 5    : 첫 보스 밤
Day 7    : 초대형 눈보라 가능성 발생
Day 10   : 두 번째 보스 + 동료 수 5~8
Day 15+  : 눈보라 주기 점점 짧아짐, 좀비 Lv 상승
Day 30+  : 엔드게임 컨텐츠 (최고 난이도, 빌드 실험)
마을 전멸 시 = 세션 종료, 점수 기록 + 메타 해금
```

## § 4. 전투 & 장비 시스템

### 4.1 하이브리드 구조

| 층 | 영속성 | 획득 |
|---|---|---|
| 기본 장비(무기·방어구·부적) | 영속 (마을 창고) | 대장간 제작, 사냥 드랍, 보스 드랍 |
| 런타임 룬/축복 | 해당 밤에만 | 낮 설원에서 룬 조각 수집, 밤 레벨업 시 선택 |
| 진화 무기 | 밤 내 일시 | 기본 무기 + 특정 룬 조합 충족 시 자동 진화 |

### 4.2 기본 장비 (영속)

**슬롯**
- 주무기 1 + 보조무기 1 + 방어구 1 + 부적 1

**무기 카테고리 (6종 초기)**

| 무기 | 공격 패턴 | 특징 |
|---|---|---|
| 롱소드 | 전방 휘두르기 | 균형, 초보 친화 |
| 창 | 전방 길게 찌르기 | 긴 사정거리, 관통 |
| 활 | 자동 조준 원거리 | 사거리 최장, 근거리 약함 |
| 서리 지팡이 | 광역 냉기 장판 | 슬로우 + 광역 |
| 워해머 | 강한 근접 슬램 | 기절, 저공속 |
| 쌍단검 | 고공속 근접 | 치명타 의존 |

**강화**
- 대장간에서 레벨 1→10, 자원 소모 증가
- 인챈트: 특수 효과 부여 (한 슬롯, 예: "화염 대미지 +30%")

### 4.3 런타임 룬/축복

- 밤 동안 좀비 처치 시 경험치 획득 → 레벨업 → 3개 중 선택
- 밤 끝나면 선택한 룬 전부 리셋
- 낮에 설원에서 룬 조각을 모으면 밤 룬 풀에 해당 카테고리 가중치 부여

**룬 카테고리**

| 종류 | 예시 |
|---|---|
| 공격 증강 | 투사체 +1, 대미지 +20%, 사거리 +15% |
| 방어 | 최대 HP +10%, 받는 피해 -10%, 블록 확률 |
| 유틸 | 이동 속도 +10%, 경험치 +20%, 픽업 반경 +30% |
| 광역 | 주변 적에 전기, 화염 장판, 얼음 폭발 |

### 4.4 진화 무기

- 기본 무기 + 특정 룬 2~3개 조합 충족 + 레벨 최대치 → 밤 중 자동 진화
- 진화 시 화려한 이펙트, 공격 패턴 변경 + 대미지 급증
- 예: 롱소드 + "연속 베기 룬" + "화염 룬" = **염화 폭풍검**
- 진화 효과는 해당 밤 종료 시 사라짐 (기본 무기만 영속)
- 초기 진화 수: 무기 6종 × 1~2개 = 10개 내외

### 4.5 동료 전투 AI

3종 스탠스 (§6에서 상세):
- 따르기, 사수, 공세

동료 장비: 무기 1 + 방어구 1 (단순화).

### 4.6 대미지 계산 (결정성)

```
최종 대미지 = 기본 공격력
  × (1 + 룬 공격 증강 합)
  × (1 + 진화 보너스)
  × 크리티컬 배율
  × 약점/저항 배율
  - 적 방어력 (감산)
```

시뮬 틱(30Hz) 고정 dt, RNG 시드 기반 → 재현 가능, 리플레이 가능.

### 4.7 성능 고려

- 투사체는 풀링 (사전 할당 1000개)
- 대미지 이벤트는 버퍼링 후 틱 마무리 일괄 적용
- 피격 숫자/이펙트 LOD (거리 기반 생략)

## § 5. 마을 시스템 & 자원

### 5.1 자원 (6종)

| 자원 | 용도 | 획득 |
|---|---|---|
| 나무 | 건물·바리게이트·연료 | 벌목 |
| 돌 | 건물·벽·종탑 | 채광 |
| 철 | 장비 제작/강화 | 심경 광맥 |
| 고기 | 전투 버프, 회복 포션, 희귀 장비 재료 | 사냥 |
| 식량 | 동료 일일 소비 (동료 1명당 1/일) | 밭, 드물게 채집 |
| 눈꽃 결정 | 인챈트·메타 해금 | 보스, 초대형 눈보라 생존 보상 |

→ **식량**은 생존 필수재, **고기**는 보너스/특수재로 역할 분리.

**초기 지급(타이트)**
- 튜토리얼 보너스 없음. Day 1부터 자원 빡빡함.
- 지급: 모닥불 1기 + 바리게이트 3~5개 분량만.

### 5.2 건물 (7종 + 바리게이트)

**그리드 배치** — 마을은 24×24 타일 그리드, 자유 배치 (최소 1칸 접근로 유지).

| 건물 | 풋프린트 | 용도 | 업그레이드 (L1→L5) |
|---|---|---|---|
| 모닥불 | 2×2 | 디펜스 핵심, 체온/시야/버프 | 반경·대미지·연료 효율 증가 |
| 창고 | 3×3 | 자원 저장 한도 | 저장 한도 100→250→500→850→1500 |
| 숙소 | 3×3 | 동료 슬롯 | 최대치 5→10→15→22→30 |
| 대장간 | 3×3 | 장비 제작/강화/인챈트 | 레시피 해금, 강화 한도 증가 |
| 밭 | 3×3 | 식량 생산 (동료 일꾼 배치) | 슬롯 1→2→3, 생산량 2→6/일 |
| 목책/바리게이트 거점 | 2×2 | 바리게이트 배치 권한/체력 | 배치 가능 수, 내구도 증가 |
| 종탑 | 2×2 | 밤 경고, 원거리 자동 사격 | 경고 정확도, 투사체 강화 |

추가 고려(스코프 여유 시): 제단, 약초밭, 망루. **MVP에서는 제외**.

### 5.3 모닥불 (디펜스 핵심)

- 반경 내 지속 화염 대미지 (좀비 DPS)
- 반경 내 플레이어·동료 시야 확장 + 공격력 버프 (+10~20%)
- 초대형 눈보라 시 반경 내 체온 유지 (누적 대미지 무효)
- 연료 소모: 나무 시간당 1~3개. 떨어지면 꺼짐.
- 업그레이드: 반경·대미지·연료 효율 ↑
- 여러 모닥불 배치 vs 하나 크게 키우기. 반경 겹치면 보너스 (동료 공속 +10%)

### 5.4 바리게이트 상세

**타입 (3종)**

| 타입 | 비용 | HP | 특수 효과 |
|---|---|---|---|
| 나무 바리게이트 | 나무 5 | 200 | 좀비 이동 차단, 공격 유도 |
| 돌벽 | 돌 10 | 500 | 더 튼튼, 부수는 데 오래 |
| 가시 함정 | 나무 3 + 철 1 | 100 | 통과 시 대미지 50, 1회성 |

**배치 / 수리**
- 건설 페이즈(저녁 전환) + 낮 시간 마을 내에서 가능
- 밤에는 배치 불가
- 수리: 자원 50% 투입 → HP 80% 즉시 복구

**좀비와의 상호작용**
- 좀비는 A* 경로 탐색 시 바리게이트를 "일부 저항이 있는 벽"으로 간주
- 우회 경로가 +N타일 이내면 우회, 그 이상이면 파괴

### 5.5 종탑

- 초대형 눈보라 경고 (종소리·하늘 어두워짐·까마귀·30초 카운트)
- L2+ 업그레이드 시 원거리 자동 사격 (우선: 강한 적/보스)
- 마을당 1개만 가능

### 5.6 건설/업그레이드 흐름

- 건물 건설: 저녁 전환 또는 낮 시간 마을 내에서만
- 업그레이드: 필요 자원 + 1일 대기 (즉시 반영 아님)
- 다중 업그레이드 동시 진행 불가 (건설팀 1개 개념)

### 5.7 밭 (Farm) 시스템

| 속성 | 값 |
|---|---|
| 크기 | 3×3 |
| 비용 (L1) | 나무 20, 돌 5 |
| 일꾼 슬롯 | L1: 1 / L2: 2 / L3~5: 3 (효율 증가) |
| 생산량 | 일꾼 1명당 L1: 2식량/일 → L5: 6식량/일 |
| 겨울 패널티 | 반경 내 모닥불 없으면 생산 50% 감소 |

**일꾼 배치**
- 배치된 동료도 전투 참여 가능 (밤에 싸움)
- 낮에 농사만 할지, 설원 탐험에 데려갈지는 플레이어 선택
- 해제하면 즉시 전투 가능, 해제 당일 밭 생산 0

**트레이드오프**
> "전투력 vs 식량 생산" — 농사 효율 높은 동료를 밭에 두면 식량 안정적, 하지만 밤 전투력은 제한적.

### 5.8 동료 타입 (주민 기반, 판타지 직업 삭제)

농사는 모두 가능, 전투는 신체 능력에 따라 제한.

| 타입 | 농사 효율 | 전투 가능 무기 | 전투력 | 스폰 빈도 |
|---|---|---|---|---|
| 아저씨 (장년 남) | 100% | 검·창·활·도끼 전부 | ★★★★ | 25% |
| 아줌마 (장년 여) | 110% | 활·창·단검 | ★★★ | 20% |
| 청년 (남/여) | 90% | 전부 | ★★★★★ | 10% |
| 노인 (남/여) | 70% | 지팡이·활 (제한) | ★★ | 25% |
| 아이 (소년/소녀) | 60% | **활·투석만** (근접 불가) | ★☆ | 20% |

**핵심 규칙**
- 근접 무기: 아저씨·아줌마(창·단검 한정)·청년만
- 원거리 무기: 모두
- 방어구: 모두

**캐릭터 구조의 재미**
- 아이 동료는 약하지만 감정적 애착 (영구 사망 무게감↑)
- 노인은 오래 못 버티지만 밭에서 노련함
- 마을이 "진짜 사는 공간" 느낌
- 설원에서 만나기 전 외형(아이·노인·장년·청년)으로 타입 판별 가능 → 영입 전략

### 5.9 튜토리얼 (인게임 컨텍스트 기반)

**T1. 오프닝 — 세계관 전달 (Day 0)**
- 짧은 컷신/일러스트 슬라이드 3~5장 + 텍스트 (60~90초)
- "끝나지 않는 겨울의 땅, 설원에 버려진 마을 하나..."
- 건너뛰기 가능

**T2. 첫 낮 — 채집 기초 안내 (Day 1 낮 시작 직후)**
- 마을 내 NPC(늙은 문지기) 대사 + 조작 툴팁
- 첫 나무·사슴 타겟 하이라이트 → 완료 시 툴팁 해제

**T3. 첫 동료 조우 — 영입 시스템 학습 (Day 1 중후반)**
- 설원 근교에 첫 동료 확정 스폰 (튜토리얼 고정)
- "F 키로 말 걸기" 툴팁 → 대화 UI → 영입/거절 학습

**T4. 종탑 울림 — 밤 시스템 학습 (Day 1 저녁 전환)**
- 저녁 전환 30초 페이즈 강제 로비 모드
- 툴팁 스텝: 바리게이트 배치 → 모닥불 설명 → 새벽까지 생존 목표
- 첫 밤은 좀비 수 약화 (난이도 램프업)

**T5. 첫 보스/초대형 눈보라 직전**
- 각 이벤트 직전 짧은 툴팁 ("Day 5에는 강한 적이 나타납니다")

**해제**: 한 번 본 툴팁은 다시 안 뜸 (세이브 영구 플래그). 옵션에서 재열람 가능.

### 5.10 좀비의 건물 파괴

**타겟팅 규칙 (우선순위)**
1. 플레이어 / 동료 (사거리 내)
2. 바리게이트 (경로 차단)
3. 건물 (식량·모닥불 우선)
4. 종탑/대장간 (보스급 좀비 우선)

**건물 HP / 파괴**

| 건물 | HP | 파괴 시 영향 |
|---|---|---|
| 모닥불 | 400 | 즉시 꺼짐. 반경 방어 효과 상실. |
| 창고 | 800 | 자원 일부 유실 (저장량 20%) |
| 숙소 | 600 | 수용 감소 → 초과 인원 유랑민 상태 |
| 대장간 | 600 | 강화/제작 불가, 수리 전까지 |
| 종탑 | 1000 | 원거리 지원·경고 상실 |
| 밭 | 300 | 일꾼 대피, 생산 정지 |
| 바리게이트(목재) | 200 | 파괴 후 자리 비움 |

**수리**
- 낮/전환 페이즈에 자원 50% → HP 80% 즉시 복구
- 완전 파괴된 건물은 재건 비용 70%

**보스 건물 파괴 특화**
- 얼어붙은 거인: 건물 우선 + 광역 대미지
- 서리 리치: 원거리에서 창고/밭 노림

**밤 방어 전략 변화**
- 핵심 건물 주위 바리게이트 두껍게
- 외곽 유인용 + 안쪽 최후 방어선 이중화
- 종탑의 "가장 강한 적 조준" = 건물 파괴 시도 보스 차단

## § 6. 동료 시스템 상세

### 6.1 영입 흐름

1. 설원에서 NPC 조우 (근교·중경·심경 랜덤 스폰, 주기적 리스폰)
2. 접근 시 F 키 → 간단 대화 UI
3. 정보 표시: 이름, 타입, 성별, 기본 스탯
4. [영입] / [거절]
5. 영입 조건: 숙소 여유 슬롯 + 식량 재고 +1일분
6. 즉시 파티 합류, 저녁 전환 시 자동 귀가

### 6.2 동료 상태 / 파라미터

```ts
interface Companion {
  id: string;
  name: string;
  type: 'child' | 'youth' | 'man' | 'woman' | 'elder';
  gender: 'M' | 'F';
  stats: {
    hp: number;
    maxHp: number;
    attack: number;
    defense: number;
    moveSpeed: number;
    farmEfficiency: number;
  };
  equipment: {
    weapon?: WeaponId;
    armor?: ArmorId;
    trinket?: TrinketId;
  };
  assignment: 'idle' | 'farming' | 'guarding' | 'following';
  stance?: 'follow' | 'hold' | 'aggressive';
  morale: number;
  injuries: number;
  killCount: number;
  daysAlive: number;
}
```

### 6.3 스탠스 & AI (밤 전투)

| 스탠스 | 행동 |
|---|---|
| 따르기 | 플레이어 반경 ~6타일 유지, 플레이어 타겟 우선 |
| 사수 | 지정 지점 반경 ~5타일 방어. 범위 밖 무시 |
| 공세 | 가장 가까운 적 추적·교전, 마을 외곽까지 나감 |

- 무기 타입에 따라 접근 거리 자동 조정 (검: 근접 / 활: 7~10타일 거리)
- 아이·노인 원거리 전용: 자동 후방 포지셔닝
- 스탠스는 밤 중에도 단축키로 전환 가능

### 6.4 장비 할당

- 대장간에서 창고 장비를 특정 동료에게 장착
- 슬롯: 무기 1 + 방어구 1 + 부적 1
- 아이는 활/투석만, 노인은 활/지팡이만 (유효성 검사)

### 6.5 부상 / 영구 사망

**영구 사망**
- 전투 중 HP 0 → 즉시 사망 → 시체 남음
- 사망 동료는 완전 영구 삭제 (기록만 남음)
- 장비는 시체 회수 시 환수 (주변 안전 시 픽업)

**부상 경고**
- HP 30% 이하 → 비명 오디오 + 화면 외곽 빨간 깜빡임
- HP 10% → 자동 후퇴 AI (모닥불 방향 이동 시도)
- 사망 시 전체 동료 사기 -10

### 6.6 사기 (Morale)

| 요인 | 변화 |
|---|---|
| 식량 부족 | -5/일 |
| 동료 사망 | -10 (전체) |
| 보스 처치 | +15 (전체) |
| 축제 이벤트 | +20 |
| 과로 (밭+전투 연속) | -3/일 |

**효과**
- 80+: 전투력 +10%
- 60~80: 정상
- 30~60: 전투력 -10%
- 30 미만: 매일 5% 확률 탈주

### 6.7 식량 자동 소비

- 매일 새벽 전환 시 `식량 -= 동료 수`
- 부족 시 무작위 동료 HP 10% 감소 + 사기 하락
- 0 연속 3일 → 탈주/사망 확률 급증

### 6.8 명예의 전당

- 영구 사망 동료: 마을 "기억의 돌"에 이름·타입·일수·킬 수·사인 기록
- 최종 전멸 시 "과거의 마을" 아카이브로 이동 → 다음 마을 메타 해금 산정

## § 7. 시야·그리드·바리게이트

### 7.1 그리드 시스템

**마을 그리드**
- 24×24 타일 (576칸)
- 타일: 32×32 픽셀 (기본) 또는 64×64 (상세 아트)
- 풋프린트: 모닥불 2×2, 숙소 3×3, 창고 3×3, 대장간 3×3, 종탑 2×2, 밭 3×3, 바리게이트 1×1

**설원 그리드**
- 자유 이동 2D 좌표 (연속)
- 자원·동물·NPC는 16×16 청크로 관리
- 마을↔설원 전환은 씬 전환 + 위치 보정

**건물 배치**
- 메뉴 → 미리보기 → 확정
- 유효성: 겹침 불가, 최소 1칸 길 유지
- 재배치: 자원 70% 환불

### 7.2 시야 시스템 (Fog of War)

**낮 (설원)**
- 플레이어 중심 원형 시야 (기본 10타일)
- 시야 밖: 반투명 회색 안개 + 희미한 실루엣
- 지나간 지역: 옅은 "기억" 레이어 — 구조물 O, 동적 오브젝트 X

**밤 (마을)**
- 시야 반경 6타일 (좁음)
- 모닥불·횃불 반경 내 밝음
- 시야 밖: 완전 검은색 (공포/긴장감)
- 좀비 진입 시 "쿵" 사운드 + 외곽 경고

**초대형 눈보라 밤**
- 시야 추가 50% 감소 (6→3타일)
- 화이트아웃 효과

**동료 시야 공유**
- 동료 주변 ~3타일이 미니맵 반영
- 분산 배치 시 마을 전체 모니터링

**구현**
- 셰이더 기반 마스크 (Phaser RenderTexture + 알파 블렌딩)
- 광원 반경 합산 → 마스크 그림
- MVP: 단순 반경 / 추후: 레이캐스트 벽 차폐

### 7.3 바리게이트 상세

§5.4 참조. 3종 (나무·돌벽·가시 함정).

**전략적 배치 예**
```
     T                       T = 종탑
  . . . .
  . S S .                    S = 돌벽
 W . F . W                   F = 모닥불
  . S S .                    W = 나무 바리게이트
  . . . .
     G                       G = 밭

     좀비 스폰 방향 ←
```

### 7.4 경로 탐색

- 그리드 기반 A* (24×24 경로 테이블 사전 계산)
- 바리게이트 변경 시 부분 재계산

- 500마리 시: 10~20마리만 실제 A*, 나머지 리더 따라감 (cohort pathing)

### 7.5 UI / HUD

**낮**
- 좌상단: 자원 현황 (나무·돌·철·고기·식량·눈꽃)
- 우상단: 미니맵 (낮에만 활성)

**밤**

**시각적 경고**

## § 8. 아트·오디오 & 메타 프로그레션

### 8.1 아트 디렉션

**스타일: 픽셀 아트, 중세 판타지 설원**
- 캐릭터: 16×24 또는 24×32 픽셀
- 타일: 32×32
- 팔레트: 32~48 색 제한, 차가운 청록·회백색 + 모닥불/피 오렌지·빨강 대비
- 레퍼런스: *Starbound, Kingdom: Two Crowns, Graveyard Keeper, Darkwood*
- 조명 중요도 높음: 2D 광원 셰이더 필수

**레이어 구성**
1. 배경 (눈밭 타일, 가벼운 패럴랙스)
2. 지형/구조물
3. 오브젝트/NPC/플레이어
4. 투사체/이펙트
5. 시야 마스크
6. UI

**애니메이션**
- 8방향 이동 (4방향도 허용)
- 공격·피격·사망 4~6프레임
- 밤 안개·눈보라 파티클
- 모닥불 펄스

**에셋 계획 (MVP)**

| 카테고리 | 초기 수량 |
|---|---|
| 플레이어 스프라이트 | 1종 |
| 동료 스프라이트 | 5 타입 × 2 성별 = 10종 |
| 적 스프라이트 | 좀비 3 + 보스 3~5 + 설원 동물 4 + 전투 적 3 = ~15종 |
| 건물 | 7종 × 5레벨 (레벨 오버레이) |
| 무기/장비 UI | 20종 |
| 타일셋 | 설원·마을 |

### 8.2 오디오 방향

**BGM**
- 낮(설원): 잔잔·고독 + 류트·하프·목관
- 밤: 어둡고 리드미컬. 보스 밤 전용 트랙.
- 초대형 눈보라: 불협화음·바람, 음악 최소화

**SFX**
- 벌목/채광 반복 히트감
- 좀비 신음 3~5패턴
- 모닥불 크래클
- 종탑 종소리 (경고 독자 음색)
- 까마귀·바람

**시그니처 사운드**
- "종소리 3타 → 하늘 어두워짐" = 초대형 눈보라 신호
- 바리게이트 파괴 크릉크릉 소리로 방어선 상태 청각 파악

**엔진**: Howler.js 또는 Phaser 내장. 공간 오디오(거리 감쇠) 필수.

### 8.3 UI 디자인 방향

- 디제틱(diegetic) 위주: 종탑·나무 안내판 등
- HUD 최소화, 가장자리 배치
- 메뉴: 양피지 테마 + 픽셀 아이콘
- 한국어 기본, 영어 대비

### 8.4 메타 프로그레션

**점수 계산**
```
Final Score = Days Survived × 10
            + Companions Recruited × 5
            + Bosses Killed × 50
            + Mega Blizzards Survived × 100
            + Resources Accumulated Peak × 0.1
            - Companions Lost × 3
```

**메타 통화: 기억의 조각 (Memory Shard)**
- 마을 최종 점수의 0.1%가 영구 통화
- 보스 처치 +1, 초대형 눈보라 생존 +3

**메타 해금 트리**

| 카테고리 | 예시 |
|---|---|
| 시작 자원 | 다음 마을 시작 시 나무·돌 추가 지급 |
| 시작 장비 | 롱소드 레벨 2로 시작 |
| 새 건물 | 예배당·망루·약초밭 L1 해금 |
| 새 룬/진화 | 고급 룬 풀 추가 |
| 보스 이해도 | 보스 패턴 힌트 UI |
| 외형/타이틀 | 플레이어 외형, 업적 타이틀 |

→ 모두 "추가 옵션" (역진행 느낌 없음, 베이스 난이도 유지).

### 8.5 기록 & 리더보드

- 로컬: 명예의 전당 (역대 마을 Top 10)
- 글로벌: Steam 리더보드 연동 (출시 시)
- 일일 챌린지 (옵션): 고정 시드 마을

## § 9. 개발 마일스톤·리스크·스코프

### 9.1 개발 단계

**페이즈 0 — 프로젝트 기반 (1~2주)**
- Vite + Phaser + TS 보일러플레이트
- ECS 라이브러리 도입
- 핵심 타입·이벤트 버스
- Tiled 맵 로딩 파이프라인
- CI

**페이즈 1 — 수직 슬라이스 (4~6주)**
목표: "1일 플레이 가능한 최소 게임"
- 플레이어 이동 + 근접 무기 1종 + 자동 공격
- 설원 근교 + 나무·돌·사슴
- 벌목·채광·사냥
- 마을: 모닥불 + 바리게이트(나무)
- 낮→밤 전환 + 좀비 웨이브 1종
- 플레이어 사망/승리 판정
- 시야 마스크 (단순 반경)

**페이즈 2 — 핵심 시스템 확장 (6~8주)**
- 동료 영입 + 스탠스 3종 + 영구 사망
- 동료 타입 5종
- 건물 7종 + L1→L3 업그레이드
- 밭 + 식량 + 사기
- 설원 구역 3단계 + 중경 적
- 룬 시스템 + 레벨업 + 진화 무기 2~3종

**페이즈 3 — 콘텐츠 폭 (8~10주)**
- 보스 3~5종 + 초대형 눈보라
- 건물 L4~L5 + 대장간 강화/인챈트
- 바리게이트 3종 + 가시 함정
- 튜토리얼 T1~T5
- 메타 해금 트리 + 기억의 조각
- 오디오 풀세트 + 픽셀 아트 최종

**페이즈 4 — 폴리시 & 출시 (6~8주)**
- UI 정제, HUD
- 밸런스 패스
- Tauri 데스크톱 빌드 + Steam 통합 (실적/리더보드)
- 한국어 + 영어 로컬라이제이션
- itch.io 웹 데모 출시
- Steam 정식 출시

**총 예상**: 약 **6~9개월 풀타임** (파트타임 9~12개월)

### 9.2 스코프 관리

**YAGNI 컷 후보**
- 동료 타입 5종 → 3종 (아이·장년·노인)
- 바리게이트 3종 → 2종 (목재 + 돌벽)
- 진화 무기 10종 → 4~6종
- 동료 사기 → 단순화 (식량만)
- 8방향 애니메이션 → 4방향
- 영어 로컬 → 출시 후 업데이트

**절대 컷하지 않는 것**
- 낮↔밤 사이클
- 시야 시스템
- 동료 영입 + 영구 사망
- 모닥불·바리게이트 핵심 방어 루프
- 영속 마을
- 초대형 눈보라 (1종이라도 필수)

### 9.3 주요 리스크 & 완화

| 리스크 | 영향 | 완화 전략 |
|---|---|---|
| 성능 (좀비 500+) | 치명적 | 페이즈 1부터 1000마리 스트레스 테스트. ECS, 풀링, flow field. |

### 9.4 테스트 전략

**단위 테스트 (Vitest)**
- 대미지 계산
- A* 경로 탐색
- 자원 경제 시뮬레이션
- 세이브/로드 직렬화

**통합 테스트**
- 시나리오: 1일 생존, 보스 처치, 초대형 눈보라 생존 → 저장/로드 무결성

### 9.5 성공 지표

- **Phase 1**: 플레이어 1명이 "1일 루프" 완주 후 "다시 하고 싶다"
- **Phase 2**: 내부 플레이어 1시간 연속 플레이. 첫 영구 사망 감정 반응.

  - itch.io: 첫 월 1000+ 플레이

### 9.6 출시 이후 로드맵 (선택)

- 업데이트 1 (3개월 후): 보스 2~3, 설원 구역 1 추가

## 부록 B — 용어집

- **근교·중경·심경**: 설원의 마을 중심으로부터의 거리 구역
