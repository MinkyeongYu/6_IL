# 64x64 Character Concept Sheets

이 폴더는 6IL의 실제 캐릭터 그래픽 기준을 `64x64` 셀로 잡기 위한 새 컨셉 시트입니다.

## Files

| File | Contents |
|---|---|
| `animals_64_concept_sheet.png` | mammoth, deer, wolf, bear, rabbit, boar: idle/action 64x64 concept cells |
| `humanoids_enemies_64_concept_sheet.png` | player, uncle companion, aunt companion, child companion, zombie: idle/action 64x64 concept cells |

## Direction

- Final runtime sprites should use `64x64` cells.
- Each character must remain readable inside one tile-sized sprite frame.
- Prefer clear silhouette and limited detail over high-resolution painted texture.
- Later production sheets should be exported as transparent PNG atlases with fixed 64x64 frame size.
- These generated sheets are concept references, not final sliced runtime atlases yet.
