작물특징
Potato기본 작물. 추위에 비교적 안정적이고 평균 수확량이 좋다.
Turnip저온 생존률이 높지만 수확량이 낮다.
Wheat수확량이 높지만 추위와 눈보라에 취약하다.

리소스 적용 기준

  • Unity 런타임 리소스는 Assets/Resources/와 Unity import .meta를 기준으로 관리한다.
  • 이미 프로젝트에 들어온 외부/승인 리소스를 우선 적용한다.
  • PowerShell System.Drawing으로 만든 저품질 절차형 아트는 새 게임 리소스로 사용하지 않는다.
  • 스프라이트가 없을 경우 단색 fallback만 남기지 않고, 가능한 기존 HUD/작물/프랍 아이콘을 재사용한다.
