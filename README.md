# 6_IL

판타지 중세 설원에서 마을을 개척하는 뱀서바이벌류 게임.

## 개발 환경

- Node 20+
- npm

## 스크립트

- `npm run dev` — Vite 개발 서버
- `npm run build` — 프로덕션 번들
- `npm run test` — 단위 테스트
- `npm run lint` — ESLint
- `npm run typecheck` — TypeScript 타입 검사

## 설계 문서

- 게임 설계서: `docs/superpowers/specs/2026-04-21-6il-game-design.md`
- 구현 플랜: `docs/superpowers/plans/`
