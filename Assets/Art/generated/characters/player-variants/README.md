# Red-Haired Player Variants

Generated protagonist sheets based on the red-haired girl survivor direction.

| File | Purpose |
|---|---|
| `player_survivor_axe_onehand_source.png` | Source generation for the small one-handed axe version |
| `player_survivor_unarmed_source.png` | Source generation for the no-weapon version |

Runtime candidates are exported to:

| Runtime file | Texture key |
|---|---|
| `public/assets/characters/player_survivor_axe_onehand.png` | `player` |
| `public/assets/characters/player_survivor_unarmed.png` | `player_unarmed` |
| `public/assets/characters/player_survivor_axe.png` | `player_axe_big` |
| `public/assets/characters/player_survivor_bow.png` | `player_bow` |

All runtime sheets are 384x384 PNG files with 4x4 frames at 96x96 each.
